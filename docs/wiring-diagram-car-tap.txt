ETHANOL CONTENT ANALYZER - IN-CAR TAP WIRING DIAGRAM
=====================================================

Tapping into an existing flex fuel sensor already installed in the vehicle.
The ECU keeps its original connection — the ESP32 eavesdrops on the signal.


VEHICLE WIRING (tap into the sensor connector)
==============================================

FLEX FUEL SENSOR                              ECU
+-----------------------------+          +-----------+
| Pin 1: Ground  (yellow)     |--+-----> |           |
| Pin 2: Signal  (orange)     |==+======>|  Signal   |
| Pin 3: 12V Power (green)    |--+-----> |           |
+-----------------------------+  |       +-----------+
                                 |
                           TAP ALL 3 PINS


NEW WIRING
==========

Signal tap:

  sensor signal (orange) ----+----> ESP32-C3 GPIO4
                             |
                             +---[R1 2.2kΩ to 3.3kΩ]----> ESP32-C3 3V3

Ground tap:

  sensor ground (yellow) ---------------------------> ESP32-C3 GND

Power tap:

  sensor 12V (green) ----> DC-DC buck converter ----> ESP32-C3 5V


ESP32-C3 SUPER MINI
+-----------------------------+
|          USB-C              |
|       +--------+            |
|  5V  <===== DC-DC VOUT+    |
|  GND <===== Common GND     |
|  3V3 -|        |- GPIO10   |
|  GP4 -|  C3    |- GPIO9    |
|  GP5 -|        |- GPIO8    |
|  GP6 -|        |- GPIO7    |
|  GP7 -|        |- GPIO6    |
|  GP0 -|        |- GPIO5    |
|  GP1 -|        |- GPIO4 <====== Signal tap (orange)
|       +--------+            |
+-----------------------------+


IMPORTANT NOTES
===============
- Do not cut the existing signal wire. Tap alongside it so the ECU still receives
  the sensor signal uninterrupted.
- Keep all grounds common (vehicle chassis, sensor, DC-DC, ESP32).
- Use a pull-up resistor to 3.3V on the ESP32 side. Do not use the old divider.
- The working wire mapping measured on this project was:
  - yellow = ground
  - orange = signal
  - green = 12V
- The firmware uses GPIO4 for the sensor signal.


PARTS LIST
==========
1x ESP32-C3 Super Mini
1x DC-DC Buck Converter module (12V to 5V)
1x 2.2kΩ or 3.3kΩ resistor (signal pull-up)
3x T-splice / solder taps (12V, GND, signal)
Wire, heat shrink, connectors as needed
